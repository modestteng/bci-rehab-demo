# 脑机协同康复机械臂闭环可视化系统

面向「揭榜挂帅」竞赛的脑机接口康复机械臂 APP 原型。整套系统在一块手机屏内完成从脑电输入到训练报告的完整闭环演示。

**在线演示：https://modestteng.github.io/bci-rehab-demo/**

---

## 系统闭环

演示流程固定为七个环节，每一环都有独立的可视化与真实的数据联动：

1. **脑电信号模拟** — 前额叶通道波形，随意图动态变化
2. **运动意图识别** — 抬手 / 伸肘 / 握拳 / 放松四类运动想象，输出置信度与误操作风险
3. **光电感知校验** — RGB / NIR / ToF 三模态交叉校验姿态与轨迹偏差
4. **AI 康复决策** — 依据意图、姿态评分、疲劳指数生成训练方案与保护策略
5. **机器人动作响应** — Three.js 程序化机械臂三维仿真，目标轨迹 / 实际轨迹 / 控制误差实时对比
6. **触觉反馈** — 压力、振动、位置感三类反馈，构成感觉运动闭环
7. **训练报告生成** — 汇总时长、完成次数、各项精度指标与康复建议

## 关键指标

| 指标 | 竞赛要求 | 当前值 |
| --- | --- | --- |
| 意图识别准确率 | ≥ 85% | 88.0% |
| 响应延迟 | ≤ 200ms | 176ms |
| 控制精度 | ≤ 5mm | 3.8mm |
| 抓握力度误差 | ≤ 10% | 7.0% |

## 双范式混合 BCI

除主动型的**运动想象**范式外，系统另设独立的 **SSVEP 稳态视觉诱发**模块，演示「被动接受式」脑机接口：

- 四个目标以 8 / 10 / 12 / 15 Hz 做频率-相位联合调制闪烁
- 枕区 O1 / Oz / O2 三通道脑电中浮现同频稳态响应
- 功率谱呈现基频与二次谐波峰，FBCCA 输出各候选频率的相关系数 ρ
- 信息传输率按 Wolpaw 公式实算：**39.2 bits/min**（N=4, P=0.95, T=2.5s）

主动范式需要患者具备残余运动想象能力；被动范式认知负担低、基本免训练，为康复早期与重度运动障碍患者提供了低负荷的备用通道。两者互补即构成混合 BCI。

> **光敏安全**：8–15 Hz 闪烁落在光敏性癫痫的风险频段（3–30 Hz），因此该模块的闪烁**默认关闭**，需访问者显式开启，并采用低对比度亮度调制，同时遵循 `prefers-reduced-motion`。

## 技术栈

React 18 + TypeScript + Vite 5，三维仿真使用 three.js 程序化建模（无外部模型文件）。全部信号均为演示用合成数据。

应用不请求任何外部资源，无字体 / 图片外链，**加载后完全离线可用**。WebGL 不可用时（例如关闭了硬件加速），三维面板会自动降级为静态说明，其余功能不受影响。

## 本地运行

```bash
npm install
npm run dev          # 开发模式
npm run dev:lan      # 同一 WiFi 下用手机访问
```

## 构建与分发

```bash
npm run build          # 静态站点 -> dist/
npm run build:single   # 单文件 HTML，双击即可离线打开
npm run build:apk      # Android APK（需 Android SDK）
```

各种分发方式的详细步骤与坑点，见 [分发说明.md](分发说明.md)。

## 深链

在网址后追加参数可直达任意画面，答辩演示用：

| 参数 | 效果 |
| --- | --- |
| `?tab=demo&step=5` | 直达「机器人动作响应」（3D 机械臂） |
| `?tab=ssvep` | SSVEP 被动接受范式页 |
